## 📌 Project Description
This project is an Open Source Software (OSS) Audit Toolkit developed using shell scripting. It is designed to analyze system information, check installed packages, audit disk usage, and generate useful reports. The project demonstrates the practical implementation of open-source concepts such as transparency, collaboration, and automation.

---

## 🎯 Objective
The main objective of this project is to:
- Understand open-source software concepts
- Build useful system-level scripts
- Automate system monitoring and auditing tasks
- Encourage collaboration and code sharing

---

## 🛠️ Technologies Used
- Linux
- Shell Scripting (Bash)
- LibreOffice (for documentation)

---

## 📂 Features / Scripts
1. **System Info Script** – Displays system details  
2. **Package Checker** – Checks installed packages  
3. **Disk Audit** – Monitors disk usage  
4. **Log Analyzer** – Analyzes system logs  
5. **Manifest Generator** – Generates system report
